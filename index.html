    Welcome

    This page offers resources for those interested in deploying Perl and Perl applications using the Chef configuration management tool

    Cookbooks

    The Perl Chef repository currently contains two cookbooks.

    • Perlbrew -- install and configure isolated, custom perls with the Perlbrew application
    • Carton -- deploy perl apps packaged with the Carton application

    If you've developed other cookbooks, please let us know and we'll link to them.

    Community

    While it's broader than Chef, the perl-devops@perl.org mailing list is a new forum for discussing Perl development and deployment issues. Subscribe by emailing perl-devops-subscribe@perl.org.
